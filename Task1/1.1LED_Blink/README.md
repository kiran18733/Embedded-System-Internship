# Task 1.1 – LED Blink

## Objective
To blink an LED with 1 second ON and 1 second OFF using delay() function.

## Components Required
- Arduino UNO
- LED
- 220Ω Resistor
- Breadboard
- Jumper wires

## Description
In this experiment, an LED is connected to a digital output pin of the Arduino. 
The LED is turned ON and OFF at a fixed interval of 1 second using the delay() function.

## Output
The LED continuously blinks with a 1 second time interval.
