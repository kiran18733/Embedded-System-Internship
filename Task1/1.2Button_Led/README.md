## Objective
To turn an LED ON when a push button is pressed and turn it OFF when released.

## Description
In this experiment, a push button is used as a digital input to the Arduino.
When the button is pressed, the input pin reads HIGH and the LED is turned ON.
When the button is released, the LED is turned OFF.

## Output
The LED glows only while the push button is pressed.
